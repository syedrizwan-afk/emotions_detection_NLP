# emotions_detection_NLP

# Install Dependencies

"pip install -r requirements.txt"

The model was fine-tuned using:

The HuggingFace Transformers library with DistilBERT/BERT as the backbone.

Epochs: 3–5

Learning Rate: 2e-5

Batch Size: 16–32

Evaluation Metrics:

Accuracy

Precision

Recall

F1-Score

Confusion Matrix

Training logs, loss curves, and evaluation plots are available in the notebook.
